# Homebrew Tap for PaperQuire

Markdown to branded PDF — desktop app with CLI.

## Install

```bash
brew tap paperquire/paperquire
brew install --cask paperquire
```

This installs the PaperQuire desktop app and the `paperquire` CLI command.

## Upgrade

```bash
brew upgrade --cask paperquire
```

## Uninstall

```bash
brew uninstall --cask paperquire
```

## About

This tap is automatically updated by CI when a new PaperQuire release is published.
For more information, visit [paperquire.com](https://paperquire.com).
